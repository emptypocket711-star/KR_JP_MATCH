ポリシー文書一覧

Hana

プライバシーポリシー

最終更新日: 2026-05-12

この文書はアプリ審査および初期リリース準備用の公開ポリシーです。正式運用前に、運営者名、サポートメール、事業者情報を実際の情報に更新してください。

1. 取得する情報

Hanaは、韓国と日本のユーザーによる言語交換および1対1チャット機能を提供するため、以下の情報を取り扱うことがあります。

  • アカウント情報: Firebase Auth識別子、メールアドレス、ソーシャルログイン識別情報
  • プロフィール情報: 表示名、生年、性別、国籍、居住国、使用言語、学習言語、自己紹介、興味キーワード、プロフィール写真
  • 利用情報: 発見画面での表示、1対1チャットルーム作成、メッセージ、翻訳状態、ラウンジ投稿・コメント・返信、ブロック、通報、評価、設定値
  • 決済およびポイント情報: 購入商品ID、購入トークンのハッシュ、ポイント付与・利用履歴、サーバー検証状態
  • 端末およびセキュリティ情報: FCMトークン、App Check状態、エラーログ、不正利用防止のための技術情報

2. 利用目的

  • 会員登録、ログイン、プロフィール設定、アカウント管理
  • 韓国語・日本語を中心としたユーザー発見、プロフィール詳細、1対1チャットの提供
  • メッセージ原文の保存および翻訳表示
  • ラウンジ投稿、コメント、返信機能の提供
  • ポイントによるチャット開始および決済検証
  • ブロック、通報、不正利用防止、コミュニティ安全管理
  • 通知配信、アプリの安定性改善、障害対応
  • 法令、プラットフォームポリシー、紛争対応に必要な記録の保存

3. メッセージと翻訳

チャットメッセージは原文を保存し、翻訳結果は原文を置き換えず別の情報として追加されます。翻訳に失敗した場合でも、原文は削除または変更されません。

4. 第三者サービス

  • Firebase Authentication, Firestore, Cloud Functions, Storage, Cloud Messaging, App Check
  • Google Play BillingおよびGoogle Play Developer API
  • Google Cloud Translation API

5. 保管および削除

ユーザーがアプリ内設定から退会を申請した場合、プロフィール、写真、通知設定、評価、ラウンジ活動情報は削除または匿名化され、アクティブな1対1チャットルームは無効化されます。通報、ブロック、決済、不正利用防止、法的対応に必要な記録は、関連法令および運営ポリシーに基づき限定的に保管される場合があります。

6. ブロックおよび通報

ブロックまたは通報が行われた場合、関連するチャットルームや表示経路が制限されることがあります。通報記録および通報者情報は、通常のユーザーには公開されません。

7. 未成年者の保護

Hanaは成人ユーザーを前提としたソーシャルマッチングおよび言語交換サービスです。未成年者保護が必要な国・地域またはプラットフォームポリシーに従い、利用制限、通報処理、有害行為の防止措置を適用することがあります。

8. ユーザーの権利

ユーザーはアプリ内設定からアカウント削除を申請できます。個人情報に関する問い合わせ、削除、訂正、利用制限の請求は、運営者が案内するサポート窓口へ提出できます。

9. お問い合わせ

サポートへの問い合わせは、GitHubリポジトリのIssueまたは運営者が別途案内するサポート窓口で受け付けます。
